 What Was Analyzed

The phishing email was analyzed based on the following points:
Sender email address spoofing
Suspicious or malicious links
Urgent or misleading language
Generic greeting
Missing official verification details

 Tools Used

Web browser
Microsoft Word / Google Docs
Free online email header analyzer (for reference)

Outcome
Through this assignment, I learned how to recognize phishing emails and understand common tactics used by attackers to trick users into revealing sensitive information.
